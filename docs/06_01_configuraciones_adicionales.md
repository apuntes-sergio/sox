---
title: Configuración avanzada de Windows Server
description:  Gestión del dominio en Windows Server. 
---


En este tema nos centraremos en las herramientas y servicios avanzados de Windows Server, esenciales para la administración de redes en entornos empresariales. Cubre desde la gestión del espacio en disco y el control de acceso, hasta la implementación de servicios clave de infraestructura y las estrategias de continuidad del negocio.

| Tema | Enfoque Principal |
| :--- | :--- |
| **Carpetas Compartidas Ocultas** | Gestión de la visibilidad y uso de los recursos compartidos de administración. |
| **Cuotas de Disco** | Aplicación de límites de almacenamiento a los usuarios en volúmenes NTFS para optimizar el espacio. |
| **Servidor DHCP** | Configuración del servicio de asignación dinámica de direcciones IP y gestión de ámbitos. |
| **Escritorio Remoto** | Habilitación y securización del acceso gráfico remoto al servidor para tareas administrativas. |
| **Red VPN (Acceso Remoto)** | Implementación de una Red Privada Virtual para asegurar las conexiones de usuarios externos a la red local. |
| **Configuración Remota RSAT** | Instalación y uso de las Herramientas de Administración Remota del Servidor desde un equipo cliente Windows. |
| **Copia de Seguridad (Backup)** | Planificación e implementación de la protección de datos y del estado del sistema. |
| **Controlador de Dominio Secundario** | Instalación de un DC adicional para garantizar la redundancia y disponibilidad del servicio de directorio. |
| **Monitorización Windows Server** | Uso del Visor de Eventos y el Monitor de Rendimiento para supervisar el estado y detectar anomalías. |
| **Servicios en Windows Server** | Administración del ciclo de vida y los tipos de inicio de los servicios críticos del sistema. |
| **Auditorías Windows Server** | Configuración de directivas de auditoría para rastrear eventos de seguridad, como intentos de acceso o cambios en archivos. |
| **Windows Update Services (WSUS)** | Despliegue centralizado de parches y actualizaciones de seguridad de Microsoft en la red. |
| **Windows Deployment Services (WDS)** | Instalación de sistemas operativos de forma remota a través de la red mediante PXE. |
| **Integración de equipos Linux en AD** | Permite que equipos Linux como por ejemplo Ubuntu puedan trabajar dentro del dominio. |
